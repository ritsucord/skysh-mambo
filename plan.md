1. 수요 보장
    내가 원하는 가격에 구매하고자 하는 수요가 여럿일 경우,
    수요 또는 사용자가 일정량을 초과할 떄 알림을 보내서
    리스크가 낮은 거래 체결이 가능하도록 유도.

2. 사용자 신뢰도 검증
    커뮤니티 사용자의 과거 반응과 실제 주문내역을 대조하여,
    메세지와 거래에 괴리가 있는 경우 스팸으로 탐지하여 타
    사용자들을 보호

3. 투자자 반응 실시간 분석
    커뮤니티 글을 실시간으로 받아와, LLM으로 글에 담긴 감정을
    분석하고 정리된 형태로 거래창에 익스텐션으로 제공

# <진짜 세부>
 (i) ai모델
  - 기존의 커뮤니티 등에서 사람들의 반응과, 실제 주식 시장의 관계를 데이터로 학습 
  (ex. 사람들이 여기서 빠져야 된다 라는 반응이 있을 경우 실제 하락은 몇 % 만큼 이루어졌는지 등)
  - 이렇게 학습된 ai 모델로, 현재 커뮤니티의 실시간 반응을 토대로 어떻게 될지 예측 결과를 내놓음 <-- 이후 익스텐션에 쓰일 부분
  - 스팸 걸러내기 가능

 (ii) 커뮤니티
  - 자신의 과거 투자 내역을 공개해야 들어갈 수 있음.
  - 투자 내역을 공개한 사람들끼리의 커뮤니티이므로 안정성이 높음
  - 수요조사를 통해 검증이 된 사람들끼리 가격 수요조사 가능
  - 동의약관에 동의한 사람들이므로 ai 학습 데이터로도 사용 가능
